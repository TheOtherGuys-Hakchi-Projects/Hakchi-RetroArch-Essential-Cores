=== Gambatte Core for RetroArch ===

Module adds support for Game Boy and Game Boy Color

Available executables and arguments to run Core:
- /bin/gambatte <rom> <clover_args>
- /bin/gb <rom> <clover_args>
- /bin/gbc <rom> <clover_args>
